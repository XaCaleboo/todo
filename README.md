# To-do list

## About

It is a pet project, based on React.js (CRA) and styled-components.

What it can do:

1. Add new tasks
2. Show you the list of your tasks
3. Delete tasks
4. Edit existing tasks

## Run on dev

```bash
git clone https://github.com/XaCaleboo/todo.git
cd todo
npm i
npm start
```

## Build

```bash
git clone https://github.com/XaCaleboo/todo.git
cd todo
npm i
npm build
```
